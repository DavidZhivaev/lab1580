# Лабораторные работы
ГБОУ «Бауманская инженерная школа №1580»
Данный репозиторий содержит лабораторные работы, выполненные в рамках учебного процесса.

## Назначение
Репозиторий используется для:
- хранения лабораторных работ;
- хранения учебных материалов;
- передаче материалов преподователю;

## ⚠️ Доступ
Репозиторий не является публичным проектом и предназначен исключительно для учебного использования.
