# Алгоритм для лабораторной первой работы.

## 1. Ввод и первичная обработка
* Считать строку, введенную пользователем.
* Разбить строку на два отдельных значения (часы и минуты).

## 2. Проверка входных данных
* Проверить, являются ли оба значения целыми числами.
* Выполнить проверку диапазонов:
    * Часы: от 0 до 23 включительно.
    * Минуты: от 0 до 59 включительно.
* Если данные некорректны, вывести описание ошибки и завершить работу.

## 3. Обработка постоянных случаев
* Если часы и минуты равны 0 — результат "полночь".
* Если часы равны 12, а минуты равны 0 — результат "полдень".

## 4. Определение времени суток
* В зависимости от значения часов, выбрать период:
    * [0, 6) -> ночь.
    * [6, 12) -> утро.
    * [12, 18) -> день.
    * [18, 24) -> вечер.

## 5. Формирование текстового описания
* Перевести часы в 12-часовой формат для отображения.
* Определить правильное окончание для слова "час" на основе текущего числа.
* Если минут больше 0:
    * Определить окончание для слова "минута".
    * Составить строку: [часы] + [минуты] + [период].
* Если минут 0:
    * Добавить слово "ровно".
    * Составить строку: [часы] + [период] + "ровно".

## 6. Вывод результата
* Отобразить готовую строку пользователю.
